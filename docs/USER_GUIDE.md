## Getting Started

### System Requirements

- **Browser**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **JavaScript**: ES2020 support required
- **Screen Resolution**: 1024x768 minimum (1920x1080 recommended)
- **Memory**: 4GB RAM minimum (8GB recommended)
- **Storage**: 100MB available space

### First Launch

1. **Open DurgasOS** in your web browser
2. **Wait for loading** - The system will initialize (3-5 seconds)
3. **Desktop appears** - You'll see the desktop with wallpaper and icons
4. **Start exploring** - Double-click icons to open applications

### Initial Setup

1. **Personalize your desktop**:
   - Right-click on desktop → "Personalize"
   - Choose your preferred wallpaper
   - Select light or dark theme
   - Adjust accent colors

2. **Configure taskbar**:
   - Right-click on taskbar → "Taskbar settings"
   - Choose taskbar position (bottom, top, left, right)
   - Enable/disable auto-hide
   - Customize system tray

3. **Set up notifications**:
   - Click the notification bell in system tray
   - Configure notification preferences
   - Set up do not disturb schedules

## Window Management

### Opening Windows

#### From Desktop Icons
1. **Double-click** any desktop icon
2. Window opens with default size and position
3. Window appears in taskbar

#### From Start Menu
1. **Click** Start button
2. **Click** on desired application
3. Window opens and appears in taskbar

#### From File Explorer
1. **Double-click** any file
2. Associated application opens the file
3. Window shows file content

### Window Controls

#### Title Bar
- **Left side**: Window icon and title
- **Right side**: Minimize, Maximize/Restore, Close buttons

#### Window Actions
- **Minimize**: Click minimize button or double-click title bar
- **Maximize**: Click maximize button or double-click title bar
- **Restore**: Click restore button (appears when maximized)
- **Close**: Click close button or press Alt+F4

#### Moving Windows
1. **Click and drag** title bar
2. **Release** to drop window
3. **Snap to edges** for automatic sizing

#### Resizing Windows
1. **Hover** over window edges or corners
2. **Click and drag** to resize
3. **Release** to set new size

### Window Snapping

#### Snap Layouts
- **Left Half**: Drag to left edge of screen
- **Right Half**: Drag to right edge of screen
- **Top Half**: Drag to top edge of screen
- **Bottom Half**: Drag to bottom edge of screen
- **Quarter Windows**: Drag to corners
- **Full Screen**: Drag to top edge

#### Snap Groups
- **Create**: Select multiple windows and right-click → "Group"
- **Ungroup**: Right-click group → "Ungroup"
- **Switch**: Click group in taskbar to cycle through windows

### Virtual Desktops

#### Creating Desktops
1. **Click** task view button in taskbar
2. **Click** "New Desktop" button
3. **Name** your desktop
4. **Switch** between desktops

#### Managing Desktops
- **Switch**: Click desktop thumbnail
- **Rename**: Right-click desktop → "Rename"
- **Delete**: Right-click desktop → "Delete"
- **Move Windows**: Drag windows between desktops

#### Desktop Shortcuts
- **Win+Tab**: Open task view
- **Win+Ctrl+Left/Right**: Switch desktops
- **Win+Ctrl+D**: Create new desktop
- **Win+Ctrl+F4**: Close current desktop

## Troubleshooting

### Common Issues

#### Performance Issues

**Problem**: Slow performance or lag
**Solutions**:
1. **Close** unnecessary applications
2. **Restart** the system
3. **Check** available memory
4. **Disable** unnecessary animations
5. **Update** browser to latest version

**Problem**: High memory usage
**Solutions**:
1. **Close** unused applications
2. **Restart** the system
3. **Check** for memory leaks
4. **Disable** unnecessary features
5. **Clear** browser cache

#### Display Issues

**Problem**: Icons not displaying correctly
**Solutions**:
1. **Refresh** the desktop (F5)
2. **Restart** the system
3. **Check** icon cache
4. **Update** browser
5. **Clear** browser cache

**Problem**: Windows not snapping properly
**Solutions**:
1. **Check** window snapping settings
2. **Restart** the system
3. **Update** browser
4. **Disable** browser extensions
5. **Check** for conflicts

#### Application Issues

**Problem**: Applications not opening
**Solutions**:
1. **Check** application registry
2. **Restart** the system
3. **Reinstall** application
4. **Check** permissions
5. **Update** browser

**Problem**: Applications crashing
**Solutions**:
1. **Check** error logs
2. **Restart** the system
3. **Update** application
4. **Check** compatibility
5. **Reinstall** application

### Error Messages

#### Common Error Messages

**"Application not found"**
- Check if application is installed
- Verify application registry
- Restart the system
- Reinstall application

**"Permission denied"**
- Check user permissions
- Run as administrator
- Check file permissions
- Verify security settings

**"Out of memory"**
- Close unnecessary applications
- Restart the system
- Check available memory
- Increase virtual memory

**"Window not responding"**
- Wait for response
- Force close window
- Restart the system
- Check for conflicts
