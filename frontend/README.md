# DurgasOS Frontend

Next.js 13+ frontend for DurgasOS Desktop Environment.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local` file:
```bash
cp .env.example .env.local
```

3. Update `.env.local` with your configuration.

4. Run the development server:
```bash
npm run dev
```

## Docker

```bash
docker-compose up
```
